# `src/data/indexes/` ‚Äî Pet Product Category Indexes

This folder contains auto-generated TypeScript index files. Each file represents one **mutually-exclusive product category** and exports an array of `{ sku, tags }` entries for every product that belongs to that category.

> **Do not edit these files manually.** They are regenerated by running:
> ```bash
> pnpm datasets:compile:pets
> ```
> Source script: `datasets4/compile-pets-products.ts`

## Entry Shape

Each `*.ts` file (except `index.ts`) exports a default array of:

```ts
interface IndexEntry {
  /** Product SKU ‚Äî matches the folder name under products/ */
  sku: string;
  /** All classification tags: bucket name + breadcrumb segments + brand + traits */
  tags: string[];
}
```

### Tag Convention

| Tag pattern | Meaning |
|---|---|
| `dog-dry-food` | Bucket name (always first tag) |
| `purina`, `hills`, `royal-canin` | Brand (lowercase, hyphenated) |
| `adult`, `senior`, `puppy` | Life stage |
| `large-breed`, `small-breed` | Breed size |
| `grain-free` | Dietary attribute |
| `kidney-care`, `liver-care` | Health condition |
| `has-ingredients` | Product has ingredient list |
| `has-nutrition` | Product has nutrition analysis |
| `in-stock`, `out-of-stock` | Availability |
| `popular`, `best-seller` | Popularity signal (AliExpress) |

## `index.ts` ‚Äî Category Manifest

`index.ts` default-exports a `CategoryIndex[]` array:

```ts
interface CategoryIndex {
  /** Filename relative to this folder, e.g. "dog-dry-food.ts" */
  filename: string;
  /** Tags that describe the content ‚Äî useful for routing/filtering UI */
  tags: string[];
}
```

Example entry:
```json
{ "filename": "dog-dry-food.ts", "tags": ["dog", "food", "dry", "kibble"] }
```

Use it to build navigation, filter panels, or sitemap entries without importing every category file.
